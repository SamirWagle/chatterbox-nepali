# Chatterbox Nepali TTS

Fine-tuned Chatterbox multilingual TTS checkpoint for Nepali text-to-speech.

## Files

- `t3_mtl_nepali_final.safetensors` - final fine-tuned T3 weights.

## Usage

Use the companion Space:

https://huggingface.co/spaces/waglesameer5/chatterbox-nepali-tts

The Space loads this model by default and asks for a clean 5-10 second reference
voice clip.
